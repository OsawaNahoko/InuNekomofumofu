ゲーム概要
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
三年生後期課題の『GameObjectをクリックできる作品』にて制作した作品です。授業としての要項以外に遊びの表現を入れて制作しました。
私のもふもふ不足を解消するために犬と猫にしました。
UnityRoomでも「癒されました。」とのコメントを頂き狙い通りで嬉しくなりました。
設計の工夫した点としては、ベースをモグラたたきにして色んなキャラクターを追加したい！という計画を立てました。
そこで親クラスを設計して、「いぬ」と「ねこ」それぞれ別のキャラクターとして追加しやすいように、またさらに他のキャラクターを実装しやすいように意識して制作しました。
この結果により、現在はいぬとねこで潜伏するまでの時間を調整出来ています。
親クラスの練習としての側面が強い作品です。
